## 使用方法

このファイルは、成功パターンと失敗パターンを記録し、今後のセッションで同じ過ちを繰り返さないようにするためのものです。

- **成功パターン**: 3回以上成功したアプローチを記録
- **失敗パターン**: 2回以上発生したエラーパターンを記録
- **試行錯誤**: 実験的なアプローチとその結果を記録
